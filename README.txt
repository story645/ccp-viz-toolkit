ccp-viz-toolkit is a gsoc-2011 project, and is a toolkit for vizualizing
climate data. The ccp-viz-toolkit intends to provide a library and 
web interface for generating climate vizualizations. 

acknowledgments:
ccp-viz-toolkit is jointly mentored by the Common Climate Project and 
the Climate Code Foundation (http://climatecode.org/) and was created 
for the Common Climate Project (CCP) and GSOC 2011. 
It is also somewhat based on code developed at http://csdirs.ccny.cuny.edu/ 

More info at: http://climatecode.org/activities/gsoc2011/aizenman/
